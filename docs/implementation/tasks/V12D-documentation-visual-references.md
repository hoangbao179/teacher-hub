# V12D — Documentation and visual references

## Goal

Đồng bộ tài liệu với sản phẩm Lớp học cô Vy và lưu ảnh chạy thật làm visual reference V2.

## Scope

- Product/features/auth/admin/user guide/UI guidelines/media replacement documentation.
- Wireframe version policy and V2 screenshot set from the running application.
- AGENTS visual-source-of-truth rules.
- Status, roadmap and known-limitations updates based on actual checkpoint results.
- Repository consistency checks for stale public/auth/visual statements.

## Out of scope

Regenerating P0 wireframes, large file moves, rewriting historical migration records and AI-generated V2 replacements.

## Verification gate

`npm run check:repo`, `npm run check:fast`, `npm run build`, then V12D verification report.
