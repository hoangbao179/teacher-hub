# V11A — Branding and public content

## Goal

Replace demo teacher and mathematics copy with source-controlled Cô Vy English-class branding while preserving all V1 business behavior.

## Scope

- Centralize teacher, brand, English grade 1–9 programs, methods, media, contact and SEO content.
- Update Homepage information architecture and restrained pastel styling.
- Synchronize metadata, manifest, sitemap, robots and public configuration validation.
- Default newly created classes and development seed classes to English examples without changing existing user data.
- Extend Homepage/configuration E2E coverage and capture the required branding screenshots.

## Out of scope

No CMS, public tuition, fabricated credentials/results, notification or lesson/tuition/schedule rule changes. No schema migration.

## Verification gate

`npm run check:fast`, `npm run test:e2e` and `npm run build` must pass. Required browser screenshots and manual visual review must be complete before V11A can be marked PASS.
