# M1.1 — Architecture stabilization

## Goal

Ổn định source-of-truth, authentication lifecycle, UI M1, database constraints,
audit foundation, local seed/reset và verification trước khi bắt đầu M2.

## In scope

- Một implementation status duy nhất và OpenAPI khớp routes.
- Token bootstrap qua `GET /api/auth/me`; xử lý 401 tập trung.
- CRUD/state transitions M1 có loading, feedback và confirmation.
- Giá gói dương; enrollment ACTIVE được DB bảo vệ; lớp paused/closed không nhận enrollment.
- Audit các mutation M1 trong transaction khi thực tế.
- Dev seed/reset, unit/integration/browser smoke và consistency check.
- Bốn ADR làm tiền đề cho M2/M3.

## Out of scope

Không triển khai lesson wizard/attendance workflow mới, late-entry recalculation,
makeup flow, import/export, notification, CMS, dependency upgrade hoặc refresh token.

Xem `.agent-reports/M1.1-implementation.md` và `.agent-reports/M1.1-verification.md`.
