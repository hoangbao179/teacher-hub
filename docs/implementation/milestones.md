# Implementation milestones

## M0 — Base (đã scaffold)

Workspace, auth, schema, migrations, contracts, API foundation, admin shell, docs/rules, Docker.

## M1 — Class/student CRUD hoàn chỉnh

Form create/edit, state transitions, one-active-enrollment guard, audit log.

## M2 — Lesson wizard hoàn chỉnh

Điểm danh, content/homework, confirm, makeup flow, mobile validation.

## M3 — Late-entry recalculation

Sửa attendance chưa paid, tái phân bổ chronological, khóa vùng paid, acceptance tests.

## M4 — Tuition UX

Detail, mark paid form, incomplete cycle, reopen paid cycle có lý do/audit nếu quyết định triển khai.

## M5 — Schedule/reconciliation

Exceptions, bulk reconcile, busy slots, conflict warning.

## M6 — Home polish + export Excel

Nội dung thật, video, SEO, Excel export theo spec.

## M7 — Production hardening

Backup, restore drill, rate limit login, observability tối thiểu, mobile QA, deployment.
